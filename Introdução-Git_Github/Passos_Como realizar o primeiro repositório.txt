Criar um conta no github através do site:https://github.com/
Acessar o repositório e criar um novo
Dar um nome ao repositório preenchendo o espaço Repository name, deixa ativado a opção Public e pode marcar a opção Add a README file, caso queira um arquivo editável.
Clica em create repository.
Você pode editar o repositório através da imagem que simboliza um lápis.
Para clonar o repositório criado para a pasta local clica em code e copia o link do repoitório.
Com a pasta local aberta abra o Git Bash Here, já instalado no seu computador.
Digite git clone + o endereço copiado. Para copiar pode ser através do clique com botão direito opção paste e enter.
Para realizar o envio das alterações da maquina local para o servidor:Basta ir no Git Bash e digitar: git commit -m "Coloque um comentário!"
Em seguida o comanda que fará o envio: git push origin main

Pronto! Repositório criado atualiazado na pasta local e depois atualizado no servidor!
